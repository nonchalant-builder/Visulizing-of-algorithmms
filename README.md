# ALGORITHM VISUALIZER

## Installation and Setup Method:

Given ZIP file in the GitHub repo contains the source code and all the necessary files to the algorithm visualizer. Extract the ZIP file to a suitable place. This Project can only be run using IntelliJ IDEA, so it is better to install it. Installation of JDK 21 is pivotal as the project was created using JDK 21. After completing the above-mentioned steps, you are good to go wild with it.

## Brief Overview of the Project:

For a comprehensive and clear understanding of algorithms, it is necessary to be able to get a feel of them. There is no better way to do so than to get a view of how they work. Algorithm visualizer was created to get this job done. It gives an overview of algorithms and showcases their working principles. It helps people to interpret the beauty of algorithms and shows the magical way an algorithm does its job.
